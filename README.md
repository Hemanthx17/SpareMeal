# 🍽️ SpareMeal

**A College Project | Share Food. Spread Hope.**

This is a web-based platform developed as part of our college curriculum to connect food donors with NGOs and individuals in need.

## 🎯 Project Objectives
- Build a responsive web application
- Learn full-stack development (Frontend + Backend)
- Implement user authentication
- Create a real-world solution for food waste reduction

## ✨ Key Features
- 🔐 **User Authentication** - Donor & NGO login system
- 🍽️ **Food Donation Posts** - Share available meals with details
- 📊 **Dashboard** - Track donations and impact
- 👤 **Profile Management** - Manage user information
- 📱 **Responsive Design** - Works on all devices

## 🛠️ Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Firebase (Database & Authentication)
- **Hosting**: GitHub Pages

## 📚 Learning Outcomes
- DOM manipulation and event handling
- REST API integration
- Database management with Firebase
- Responsive UI/UX design
- Version control with Git

## 🚀 How to Use
1. Open `index.html` in your browser
2. Sign up as a Donor or NGO
3. List or browse food donations
4. Connect and make a difference
